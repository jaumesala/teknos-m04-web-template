Plantilla base

Aquesta plantilla s'encarrega de processar arxius less i sass, importar moduls npm, concatenar i minificar codi javascript.

Codi

  • Organitza el teu codi SCSS dins del directori assets/sass. El resultat es copiarà a www/css/main.css
  • Organitza el teu codi JS dins del directori assets/js. El resultat es copiarà a www/js/main.js
  • Posa les teves tipografies dins del directori assets/fonts. Aquestes es copiaran a www/fonts
  • Dins l'arxiu gulpfile.js pots afegir moduls dins l'array npmModules. Aquests seran concatenats i copiats a www/js/plugins.js

Funcionament

  • Obre la línia de comanda (cmd + r) i executa "cmd"
  • Navega fins a la carpeta del projecte
  • Executa npm install --global gulp-cli per instalar el paquet gulp
  • Instal·la les dependències especificades a l'arxius package.json executant npm install --no-bin-links
  • Executa el gulp (hauràs d'utilitzar la ruta fins al binari ex: D:\Users\jaume.sala\AppData\Roaming\npm\gulp (Pensa a canviar el nom d'usauri)
  • Per tal de no haver d'executar el gulp cada cop que fem un canvi a un arxiu (css/js), podem executar la comanda "watch", la qual estarà pendent de canvis en els nostres arxius. Executa D:\Users\jaume.sala\AppData\Roaming\npm\gulp watch
